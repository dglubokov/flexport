# flexport
Flexport is Distributed Web-Based File Manager

![](demo.mp4)

## Development

### Frontend

**Install dependencies**

```bash
cd frontend
npm install
```

**Run development server**

```bash
npm run dev
```

### Backend

**Install dependencies**

```bash
cd backend
pip install -r requirements.txt
```

**Run development server**

```bash
uvicorn main:app --host 0.0.0.0 --port 8009 --reload
```
